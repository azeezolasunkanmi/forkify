## Overview

### The Application

Users should be able to:

- Search for recipes
- Add a recipe
- Check all the ingredients for selected recipe
- Bookmark a recipe
- See directiopns on how to cook a recipe
- See time needed to prepare one recipe
- Update nuber of servings
